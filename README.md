# MovieApp


### About
 IMDB clone. Internet movie database using Angular and [TMDB](https://developer.themoviedb.org/docs) API. Width full authentication using Firebase.
 `Features:`
 * Search Movie
 * Watch about Movie (the budget, date release, genres etc)
 * Add to favorites
